# Popular Shows & Films tvOS

Uses new tvOS and UICollectionView and customCell for popular Tv Shows and Films


Requirements
---- 
Xcode 8.0+

Swift 3

Deployment Target = iOS 10

Base SDK = iOS 10

UITapGestureRecognizer

UITraitCollection

Image Focus

Icon Template
